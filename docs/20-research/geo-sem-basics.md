# GEO / SEM 基础

更新时间：2026-06-01

## 目的

本页用于建立广告 Agent 方向的业务基础。当前只沉淀通用学习框架，不编造外部资料来源，也不记录公司内部实践细节。后续补充时应优先引用广告平台官方文档、公开产品文档或公开行业资料。

## 概念边界

### SEM

SEM 在本知识库中主要指搜索广告相关工作，包括关键词、广告组、创意、落地页、出价、预算、转化追踪和效果优化。广告 Agent 需要把业务目标转成可执行的搜索广告结构，并能根据指标反馈提出优化建议。

### GEO

GEO 在本知识库中按生成式搜索和 AI 搜索可见性方向理解，重点关注内容如何被生成式答案、搜索结果摘要、问答式搜索体验识别和引用。后续需要结合公开资料继续确认不同公司或团队对 GEO 的具体定义。

## SEM 学习主线

| 主题 | 需要理解的问题 |
| --- | --- |
| 账户结构 | Campaign、Ad Group、Keyword、Creative、Landing Page 如何组织 |
| 搜索意图 | 用户查询背后的商业意图、信息意图、品牌意图如何区分 |
| 关键词 | 关键词拓展、匹配方式、否定关键词、词包分组如何影响投放 |
| 创意 | 标题、描述、卖点、行动号召、落地页一致性如何影响点击和转化 |
| 预算与出价 | 预算上限、出价策略、消耗节奏和成本目标如何约束动作 |
| 转化追踪 | 表单、注册、购买、咨询等转化事件如何定义和回传 |
| 指标诊断 | CTR、CVR、CPA、CPC、ROI、展示量、点击量如何联动分析 |

## GEO 学习主线

| 主题 | 需要理解的问题 |
| --- | --- |
| 内容可见性 | 内容能否被搜索引擎、AI 搜索或问答系统发现、理解和引用 |
| 语义覆盖 | 品牌、产品、场景、问题、对比、评价等语义资产是否完整 |
| 可信度信号 | 来源、结构、引用、更新频率、作者或品牌可信度如何影响呈现 |
| 问题地图 | 目标用户会问什么问题，哪些问题会触发商业转化 |
| 内容结构 | FAQ、对比页、产品说明、案例、教程如何服务生成式答案 |
| 效果衡量 | 曝光、引用、品牌查询、站点流量、转化线索如何评估 |

## 广告 Agent 需要补齐的能力

- 把业务 brief 拆成关键词、问题、用户意图和素材方向。
- 识别平台规则、品牌约束和禁用表达。
- 生成可审阅的素材和投放结构草案。
- 根据指标变化诊断问题，而不是只生成泛化建议。
- 在预算、账户、合规和品牌风险上设置人工确认点。
